SQL injection UNION attacks

Όταν ένα website είναι ευάλωτο σε SQL injection και τα αποτελέσματα του ερωτήματος επιστρέφονται στις απαντήσεις της εφαρμογής, μπορείτε να χρησιμοποιήσετε τη λέξη-κλειδί UNION για να ανακτήσετε δεδομένα από άλλους πίνακες στη βάση δεδομένων. Αυτό είναι κοινώς γνωστό ως επίθεση SQL injection UNION.

Προσδιορισμός του αριθμού των απαιτούμενων στηλών

1η Μέθοδος: Order/Group by

Για να προσδιορίσετε τον αριθμό των στηλών σε ένα ερώτημα, προσαρμόστε σταδιακά τον αριθμό που χρησιμοποιείται στις προτάσεις ORDER BY ή GROUP BY έως ότου ληφθεί μια ψευδής απάντηση. Παρά τις ξεχωριστές λειτουργίες των GROUP BY και ORDER BY εντός της SQL, και οι δύο μπορούν να χρησιμοποιηθούν με τον ίδιο τρόπο για τον προσδιορισμό του αριθμού στηλών του ερωτήματος.

1' ORDER BY 1--+    #True
1' ORDER BY 2--+    #True
1' ORDER BY 3--+    #True
1' ORDER BY 4--+    #False - Query is only using 3 columns

2η Μέθοδος: UNION SELECT

Η δεύτερη μέθοδος περιλαμβάνει την υποβολή μιας σειράς UNION SELECT payloads που καθορίζονται με διαφορετικό αριθμό null τιμών.

1' UNION SELECT null-- - Not working
1' UNION SELECT null,null-- - Not working
1' UNION SELECT null,null,null-- - Worked
