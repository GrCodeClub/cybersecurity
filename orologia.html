Ορολογία


Όρος Περιγραφή
Firewall Τείχος Προστασίας Ένα πρόγραμμα ή συσκευή που αποκλείει ανεπιθύμητη πρόσβαση σε ένα δίκτυο ή υπολογιστή.
Κρυπτογραφία Cryptography Η επιστήμη της κρυπτογραφίας ασχολείται με την προστασία των δεδομένων με τη χρήση μαθηματικών αλγορίθμων.
Ταυτοποίηση Authentication Η διαδικασία επιβεβαίωσης της ταυτότητας ενός χρήστη πριν του επιτραπεί η πρόσβαση σε ένα σύστημα.
Εξουσιοδότηση Η διαδικασία που καθορίζει τις ενέργειες που ένας εξουσιοδοτημένος χρήστης μπορεί να πραγματοποιήσει σε ένα σύστημα.
Επιθέσεις DDoS Distributed Denial of Service
Επιθέσεις στις οποίες προσπαθούν να υπερφορτώσουν έναν server ή ένα δίκτυο με αιτήσεις, προκαλώντας την αναστολή των υπηρεσιών.
Phishing Ηλεκτρονικό "ψάρεμα" Μια μορφή επίθεσης κατά την οποία οι κακόβουλοι δράστες προσπαθούν να αποκτήσουν ευαίσθητες πληροφορίες, όπως κωδικούς πρόσβασης, προσωπικά δεδομένα ή πιστωτικές κάρτες, προσδοκώντας την συνεργασία των θυμάτων.
Malware AttackΕπίθεση Κακόβουλου Λογισμικού Η εισβολή σε συστήματα μέσω κακόβουλου λογισμικού με σκοπό την πρόκληση ζημιάς ή την παρακολούθηση των δραστηριοτήτων.
Malicious Links Attack Επίθεση με Χρήση Κακόβουλων Συνδέσμων Οι επιθέσεις που στοχεύουν στην παραπλάνηση των χρηστών μέσω κακόβουλων συνδέσμων, συνήθως μέσω email ή κοινωνικών μέσων.
Zero-Day Ευπάθεια Zero-Day Vulnerability Η zero-day είναι μία ευπάθεια στον κώδικα κάποιου λογισμικού. Όταν κάποιος βρίσκει έναν τρόπο να σπάσει αυτόν το κώδικα μέσω μίας ευπάθειας, λεμέ ότι βρήκε ένα zero-day.
Exploitation Εκμετάλλευση Η χρήση μιας ευπάθειας για να επιτευχθεί πρόσβαση σε έναν υπολογιστή ή σύστημα.

Event Logging Καταγραφή Συμβάντων Η διαδικασία καταγραφής συμβάντων και δραστηριοτήτων σε ένα σύστημα ή δίκτυο για σκοπούς ασφαλείας και ανάλυσης.
